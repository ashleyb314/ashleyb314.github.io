3D Model Viewer

Summary: This program loads any OBJ file onto the user's screen and lets them perform transformations on it, such as scaling and rotating.

Watch Presentation

For this project, I used Java to parce an OBJ file of vertices and faces, then drew them on the user's screen. From there, I defined different Matrix and Vector classes. After applying translations and transformations on all the vertices, I redrew the new image with the updated coordinates. I used different key presses to trigger the movement.

Sudoku Solver with Computer Vision

Summary: This program solves sudoku puzzles from a photo, then displays the solved result back onto the same image.

View Slideshow

For this project, I used OpenCV's Python library to take a sudoku puzzle and break it down into its "cells". Then, I used Pytesseract to correctly identify the number in each cell and take the input, and put it into an array that I could run through my sudoku solver algorithm. The algorithm I used was a depth-first search. From there, I took the results and displayed them back onto the image. Depending on the puzzle, the whole process takes a few seconds on average, with the digit recognition taking up the most time.

Handwritten Digit Detection with Neural Networks

Summary: I built a neural network and trained it to recognize handwriten digits.

I built a neural network and trained it to recognize handwritten digits. I used a file with numbers 0-255 showing the brightness of each pixel in the image. I wrote a program to define errors and tests then I would use to train the model. From there, I trained the program using different weights and ran tests to see if the program was getting better or worse each time and adjusted accordingly. Unfortunately, the recognition kept getting worse instead of better.

Online Blackjack

Summary: A game of blackjack against the computer that keeps score.

I wrote a game of blackjack against the computer that keeps score. I used images I found online of a deck of cards and assigned a numerical value to each card so it would be easy to total the sum of cards. Then I displayed that number to the user and presented them with the option to take another card or not. Whoever went over 21 first would lose and 1 would be added to the winner's score.
